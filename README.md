# CS-255
System Analysis and Design

Briefly summarize the DriverPass project. Who was the client? What type of system did they want you to design?

A: The DriverPass project is a way for peple to get a better understanding of driving, with the use of hands on training and coursework to help prepare for the exam. User can choose the package that suits their needs, they will be assigned to a driver when their payement is verified and their appointment choosen is confirmed. Note, each appointment made can be modified or canceled by user if requested. The owner of DriverPAss or Liam, wants the system to be secure, easy to modify packagaes by administrators, and have quick resolution for needs like resetting passwords. Driverpass is a cloud based system but has offline document viewing by downloading the requested documents. All coursework is DMV compliant and when DriverPass is notified about any changes, it updates to meet the changes made by the DMV. It is avaible on multiple platforms and Operating systems.
